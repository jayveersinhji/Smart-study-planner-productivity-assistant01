SmartStudy is a frontend-only web application designed to help students
plan their studies, track course progress, and manage daily learning tasks
from a single dashboard.

## Features
- Clean landing page
- Student dashboard
- Course progress tracking
- Visual progress bars
- Study planner (daily / weekly)

## Tech Stack
- HTML5
- CSS3 (no frameworks)
- Google Fonts

## How to Run
1. Download the project
2. Open index.html in any browser
3. Click "Get Started" to open the dashboard

## Notes
This project is intentionally kept frontend-only for simplicity
and hackathon demonstration purposes.
